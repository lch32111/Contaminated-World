Attack : Left Click
Change Attack Style : Keyboard 'B' (short attck <-> long attack)
# when you use the long attack, click the point where monsters are.

Left Move : Keyboard 'A'
Right Move : Keyboard 'D'

Jump : Keyboard 'Space bar'
Run : Keyboard 'Left Shift'

If you die, you can't write your record.
If you clear all stages, you can write your record and thought.

The record score will be calculated like this : the bonus items + the items from monsters

If you feel this game is hard, upgrade the attack ability. 
It will make it easier to clear the game.

Enjoy!
